# Release notes

The **[GitHub Releases](https://github.com/spdermn02/TouchPortal-HardwareMonitor/releases)** page is the canonical source for release notes and downloads (`.tpp`).

The files in this folder are an in-repo archive of the notes for reference. New releases don't need a file here — write the notes directly on the GitHub Release. If you want an archived copy, drop a `X.Y.Z.md` in this folder.
